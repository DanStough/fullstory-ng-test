Progression

I wanted to build a front-end only app for this project, but the FullStory API does not allow CORS access. This would have made it possible to host on GitHub pages, which would have made delivery simple. I switched to serving the app with express.js and also using it as a proxy to make API requests to FS and GitHub. This also makes the application easier to use/more secure, as the proxy layer now stores all of the API tokens.

I looked at a bunch of other npm packages to make development easier, but they were either not well suited for ES2015 modules, incomplete and/or overkill:

  • fullstory
  • ng-fullstory
  • octokit/rest.js

The application is build into a publicly available docker container (dstough/fullstory-challenge). Hosted on ECS w/ Fargate in AWS.
